* let/const

const是常量，let是变量

* 多行字符串/模板变量

使用反引号``

```
const name = 'a', age = 20
const html = `<div>
                <p>${name}</p>
                <p>${age}</p>
             </div>`
```

* 解构赋值

```
const obj = {a: 10, b:20, c:30}
const {a, c} = obj
console.log(a, c)

const arr = ['xxx', 'yyy', 'zzz']
const [x, y, z] = arr
console.log(x, y, z)
```

* 块级作用域

```
for (let i = 0; i < length; i++>) {..}
console.log(i)
```

* 函数默认参数

```
function (a, b = 10) {..}
```

* 箭头函数

改变箭头函数内的this对象，当前词法作用域覆盖了this值。

```
function fn() {
    console.log('real', this)   // {a: 100}
    var arr = [1, 2, 3]
    // 普通js  
    arr.map(function (item)) {
        console.log('js', this) //  window
        return item+1
    }
    // es6箭头函数
    arr.map(item => {
        console.log('es6', this)  // {a: 100}
        return item+1
    })
}
fn.call({a: 100})
```
