# ES6 Class

## 比较构造函数

构造函数.prototype

## Class基本语法

```
class Name {
    constructor(x, y) {
        this.x = x
        this.y = y
    }
    add() {
        return this.x + this.y
    }
}
const m = new Name(1, 2)
console.log(m.add)
typeof Name  // "function"
Name === Name.prototype.constructor  // "true"
```
语法糖形式，本质与使用Function.prototype形式相同

模仿java c#
